# API Reference

This document lists every API route with a live request you can run. Use these when the Doclific server is running (e.g. via `doclific serve`). Paths are relative to the app origin.

<HttpRequest method="GET" url="/api/codebase/file" headers="[]" queryParams="[{&#x22;key&#x22;:&#x22;filePath&#x22;,&#x22;value&#x22;:&#x22;README.md&#x22;,&#x22;enabled&#x22;:true}]" bodyType="none" bodyContent="" formData="[]" auth="{&#x22;type&#x22;:&#x22;none&#x22;}" response="">
